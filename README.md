# 安装Python包
安装HTTP库***requests***用于API接口访问
``pip install requests``

安装json库***json***用于解码JSON对象
``pip install json``

安装.shp文件处理库***shapefile***用于导出.shp矢量文件
``pip install pyshp``

安装数据分析库***pandas***用于读取数据文件
``conda install pandas``
